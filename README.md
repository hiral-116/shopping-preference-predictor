## 📌 Project Overview

In modern retail ecosystems, understanding **how and where customers prefer to shop** is critical for driving revenue, improving customer experience, and optimizing marketing strategies.

This project builds an **end-to-end machine learning system** that predicts whether a customer prefers:

- 🏬 **In-Store Shopping**
- 🛒 **Online Shopping**
- 🔄 **Hybrid (Both Channels)**

Using demographic, behavioral, and psychological features, the model learns hidden patterns in customer decision-making and converts them into actionable business insights.

The solution is designed as a **real-world retail analytics pipeline**, not just a classification model.

---

## 📊 Dataset Overview

The dataset used in this project contains customer-level behavioral and demographic attributes used to predict shopping preferences (Store, Online, or Hybrid).

It includes features such as:
- Age, Gender  
- Monthly Income  
- Internet Usage  
- Social Media Activity  
- Online & Offline Spending Behavior  
- Brand Loyalty Score  
- Impulse Buying Tendency  
- Shopping Preference


---

## 🔬 Project Pipeline

- Data cleaning and duplicate removal  
- Handling missing values  
- Encoding categorical variables  
- Feature scaling using StandardScaler  
- Splitting dataset into training and testing sets  
- Training multiple classification models  
- Evaluating performance using accuracy and confusion matrix  
- Selecting the best-performing model  
- Building prediction system for real-time inference
